## ARREGLOS UNIDIMENSIONALES

Un arreglo es una estructura de datos que almacena una coleccion de elementos del mismo tipo
bajo un mismo nombre

CARACTERISTICAS

Tamaño fijo (se define)
Acceso por indice - [1,2,3,4,5,6]
Almcena elementos del mismo tipo
Se declara con corchetes - lista, colecciones, tuplas

//Declaracion
tipo[] nombreArreglo;

//Creacion
nombreArreglo = new tipo[tamaño]

//Declaracion y crearlo en una linea
tipo[] nombreArreglo = new tipo[tamaño]

//Declaracion con elementos definidos
tipo[] = nombreArreglo = {valor1, valor2, valor3, ... valorN}
